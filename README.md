# p5.js Game 

Simple server-client browser game made using p5.js and web sockets.

## Getting Started

#### Dependencies

* ```npm install``` 

#### Execution

* ``` node server.js ```
* Open http://localhost:3000

## Usage
* WASD to move
* Scroll to switch weapons
* Click to shoot
* R to reload
* Live demo at: https://murmuring-retreat-12672.herokuapp.com/
